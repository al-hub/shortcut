# shortcut

## intellij code 작성  
main                  : live template 의 이해     
  
sht + /               : 주석 처리/해제
  
sht + enter           : 아래 엔터  
ctl + sht + enter     : { } 삽입 엔터 ★

ctl + alt + t         : try - catch ( surround with )

ctl + d               : 현재 라인 복사
ctrl + sht + up/down  : 현재 라인 코드위아래 이동
