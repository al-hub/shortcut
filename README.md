# shortcut

## intellij code 작성  
main                  : live template 의 이해     
  
shift + /               : 주석 처리/해제
  
shift + enter           : 아래 엔터  
ctrl + shift + enter     : { } 삽입 엔터 ★

ctrl + alt + t         : try - catch ( surround with )

ctrl + d               : 현재 라인 복사  
ctrl + shift + up/down  : 현재 라인 코드위아래 이동
